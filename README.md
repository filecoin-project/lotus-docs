# Lotus docs 

_That's right, we're bringing it back._

## Quickstart

1. Command your command-line to run some commands:

    ```shell
    git clone git@github.com:filecoin-project/lotus-docs
    cd lotus-docs
    npm install
    npm run start
    ```

1. View the site on [`localhost:1313`](https://localhost:1313).
