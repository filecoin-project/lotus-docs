Start the Lotus daemon process.

```shell
lotus daemon command [command options] [arguments...]
```

## stop

Stops the running daemon.

```shell
lotus daemon stop
```

| Name | Option(s) | Description | Default |
| --- | --- | --- | --- |
| Help | `--help`, `--h` | Shows the help options. | `false` |
